tsconfig.json for 5minlab

## installation

```sh
npm install @5minlab/tsconfig --save-dev
```

## usage

tsconfig.json

```json
{
  "extends": "./node_modules/@5minlab/tsconfig/tsconfig.json"
}
```

## publish

```sh
$ npm publish --registry  https://registry.npmjs.org/
```
